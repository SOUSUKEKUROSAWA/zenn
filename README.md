# Zenn

## How to make and publish articles

- `npx zenn new:article`
- `npx zenn preview`
- [emoji](https://getemoji.com)
- `published: true`
- `git add .`
- `git commit -m "~"`
- `git push origin main`

[📘 Learn more](https://zenn.dev/zenn/articles/zenn-cli-guide)

## Template of the article

```md
⚡️: 他の記事に書いてなさそうだったこと

## ✅ この記事の目的

## ❌ この記事に書いてないこと

## ℹ️ 参考

```
