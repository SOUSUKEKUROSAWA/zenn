# How to make and publish articles
- `npx zenn new:article`
- `npx zenn preview`
- https://getemoji.com/
- `published: true`
- `git add .`
- `git commit -m "~"`
- `git push origin main`

[📘 Learn more](https://zenn.dev/zenn/articles/zenn-cli-guide)

# Template of the article
```md
# ゴール
# 書いてあること
# 書いてないこと
# 参考文献
```
